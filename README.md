# Steps to install
### 1) setup .env populate fields like db user and password
### 2) Run ```composer install```
### 3) Run ```php artisan migrate ```
### 4) Run```php artisan serve```
### 5) Run ```npm install```
### 6) Run ```npm run dev```


# To run queue
```php artisan queue:work```

# To run scheduled commands
```php artisan schedule:run```
